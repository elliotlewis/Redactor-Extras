Craft ships with the following Redactor plugins:

  • Fullscreen editing 'fullscreen'
  • View/edit HTML source 'source'
  • Table 'table'
  • Page break 'pagebreak'
  • Video 'video'

To use a plugin edit an existing config or create a new config json file as per Craft's documentation.

The plugins that ship with Craft are located: craft/app/resources/lib/redactor.

    
    {
        "plugins": ["fullscreen", "source", "table", "pagebreak", "video"]
    }
